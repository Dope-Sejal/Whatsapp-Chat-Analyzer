# WhatsApp Chat Analyzer


Drop an exported WhatsApp chat (.txt) into the Streamlit app and get interactive analyses: message counts, word cloud, emoji usage, links shared, sentiment, heatmaps, top words, media counts and more. 


## Setup


1. Create a virtual environment (recommended)


2. Install dependencies

   steps to run the project
   -In  vs code open terminal and run command
   ***--  pip install -r requirements.txt
   ***--  streamlit run app.py
